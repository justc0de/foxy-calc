foxy-calc
=========

Calculator addon for Firefox

## Features
- Intuitive UI
- Pipes user input to a [flexible expression parser](https://github.com/josdejong/mathjs/blob/563b3767e6ae2e10c055da2916051e5ab70fd6eb/docs/expressions.md)
